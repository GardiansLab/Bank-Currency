# Bank-Currency
is a web crawler using banquemisr website to get the price of the Egyptian pound in other exchanges including USD,ERU, and many other currencies.

#install
1-first you need to install python3.6 [Here]( https://www.python.org/downloads/).<br>
2-downlad the files from this repo then using you command go to the Folder containing these files.<br>
3-in the command line type 
```
python scrape.py
```` 
and boom the command filled out with the current if you don’t like it you will find a file called `Bank-Currency.txt` with all the same data that appeared before in the console.<br>
